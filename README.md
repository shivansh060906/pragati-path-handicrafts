# How to edit / add / remove items from the website

---

## Homepage

- In the "content" folder, find the "homepage" folder.
- The folder will have a file named "content.md"
- Change or add or remove text from that file 
- Your changes will reflect on the site on redeployment


## About

- In the "content" folder, find the "homepage" folder.
- There will be 3 files "background.md", "handicraft.md" and "project.md"
- Change or add or remove text from that file
- Your changes will reflect on the site on redeployment

## Gallery

- In the "content" folder, find the "homepage" folder.
- For every new item, you need to create a new folder.
- The folder name should be exactly "item-x"
- x should the number of the item
- In the item folder, create 1 file: data.md
- Add the item details in that file exactly according to the following format:

```
---
title: Handmade Vase
description: A beautiful handcrafted vase.
artist: Ms. XYZ
---
```

- Add the image in the same folder in .jpg format(not .jpeg or .png)
- The name of the image file should be "image.jpg"
